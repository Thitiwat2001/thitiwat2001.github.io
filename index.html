OOP (Object-Oriented Programming)

OOP หรือ Object-Oriented Programming (การเขียนโปรแกรมเชิงวัตถุ) เป็นแนวคิดที่ใช้ในหลายภาษาโปรแกรม ซึ่งมีหลักการเหมือนกัน เช่น การใช้คลาส (Class), ออบเจ็กต์ (Object), การสืบทอด (Inheritance), การห่อหุ้ม (Encapsulation), และการโพลีมอร์ฟิซึม (Polymorphism) แต่การใช้งานในแต่ละภาษาจะมีลักษณะเฉพาะตัวที่แตกต่างกันไป

หลักการสำคัญของ OOP

  1. การห่อหุ้ม (Encapsulation) คือการซ่อนรายละเอียดการทำงานภายในของออบเจ็กต์และอนุญาตให้เข้าถึงหรือแก้ไขข้อมูลเฉพาะผ่านเมธอด (methods) ที่ถูกออกแบบไว้เท่านั้น
  2. การสืบทอด (Inheritance) คือการที่คลาสลูก (Subclass) สามารถสืบทอดคุณสมบัติและพฤติกรรม (methods) จากคลาสพ่อ (Superclass) ได้ โดยไม่จำเป็นต้องเขียนโค้ดซ้ำ
  3. โพลีมอร์ฟิซึม (Polymorphism) คือความสามารถในการใช้เมธอดเดียวกันหรืออินเทอร์เฟซเดียวกันทำงานกับออบเจ็กต์หลายประเภท
  4. แอบสแตรคชัน (Abstraction) คือการซ่อนรายละเอียดที่ไม่จำเป็นของการทำงานภายใน และแสดงเฉพาะสิ่งที่ผู้ใช้ต้องรู้หรือใช้งานเท่านั้น

เปรียบเทียบการเขียนโปรแกรม OOP ด้วย Java, C++, Python

Java C++ Python
ภาษาที่คอมไพล์เป็น bytecode และรันบน JVM (Java Virtual Machine) ภาษาที่คอมไพล์เป็น machine code โดยตรง ภาษาที่ตีความ (interpreted language)
ใช้ garbage collection จัดการหน่วยความจำ ต้องจัดการหน่วยความจำเอง ใช้ garbage collection เช่นเดียวกับ Java

ความเหมือนและความแตกต่างของ Java, C++, Python

ความเหมือน: ทั้งสามภาษาเป็นภาษาที่สนับสนุนการเขียนโปรแกรมเชิงวัตถุ (OOP) และใช้แนวคิดหลักเดียวกัน เช่น การใช้คลาสและออบเจ็กต์, การห่อหุ้มข้อมูล (Encapsulation), การสืบทอด (Inheritance), และการใช้โพลีมอร์ฟิซึม (Polymorphism)

ความแตกต่าง:

  • Java: คอมไพล์เป็น bytecode และรันบน JVM ทำให้สามารถรันได้บนหลายแพลตฟอร์ม
  • C++: คอมไพล์เป็น machine code โดยตรง ทำให้ทำงานได้รวดเร็วกว่าในบางกรณี
  • Python: เป็นภาษาที่ตีความ ทำให้เขียนโค้ดได้ง่ายและยืดหยุ่น แต่ประสิทธิภาพอาจต่ำกว่า

ตัวอย่าง Code

การประกาศคลาสใน Java, C++, Python:

Java:
class ClassName { }

C++:
class ClassName { };

Python:
class ClassName:

การสืบทอดใน Java, C++, Python:

Java:
class SubClass extends SuperClass { }

C++:
class SubClass : public SuperClass { }

Python:
class SubClass(SuperClass):
